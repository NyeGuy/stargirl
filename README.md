# stargirl
an open source space adventure for game dev star heroes
- pixelorama
- godot engine
  
# installation
- git clone the repo
- Open the "Engine" folder and launch project.godot
- Press play
- save the galaxy Stargirl!

# how to play
- arrows up and down to move
- space bar to shoot
- how many aliens can you blow up?
